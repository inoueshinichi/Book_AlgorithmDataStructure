再帰関数の同じ計算をする無駄を省くためのメモ化(キャッシュ)    
`memo[v] <- fibo(v)`の答えを格納(未計算時は-1を格納)  
フィボナッチ数列を計算する再帰関数はメモ化をすることで,  
O(((1+√5)/2)^N) -> O(N)に計算コストを劇的に削減できる.
